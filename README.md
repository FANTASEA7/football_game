# ⚽ 足球生涯 - 第一人称足球模拟游戏

从青训到巨星，用每一次选择书写你的足球传奇！

## 🎮 游戏玩法

1. **选择位置** - 前锋 / 中场 / 后卫 / 守门员
2. **青训生涯** - 经历 U12 → U15 → U19 三个青训阶段
3. **比赛决策** - 每阶段3场比赛，每场比赛通过关键决策改变局面
4. **转会市场** - 每阶段结束后可选择转会到不同球队
5. **职业结局** - 根据三阶段综合评分决定最终签约的球队

## 🏆 联赛系统

- 中甲联赛
- 中国足球超级联赛（中超）
- 英超 / 西甲 / 德甲 / 意甲 / 法甲

## 🎯 评分系统

- 比赛评分：1.0 - 10.0
- 综合评分：U12(15%) + U15(35%) + U19(50%)
- 评分越高，可签约的球队越强！

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可游玩，无需任何服务器或依赖。

- 纯 HTML + CSS + JavaScript
- 单文件架构，易于修改和版本控制
- 响应式设计，支持手机和电脑
